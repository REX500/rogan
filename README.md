Rogan npm package that returns random Joe Rogan quotes.

# How to use?

* first run the npm install command  
`npm i --save rogan`
* require the package  
`const rogan = require('rogan');`
* run the getQuote() function that will return a unresolved promise containng a quote  
`rogan.getQuote()`
